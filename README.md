# ⚡ UGC Script Engine: AI-Powered Ad Framework

Most AI-generated ads sound like 1990s infomercials. This framework forces LLMs to adopt a **"Creator-First"** persona to generate scripts that actually convert on TikTok, Reels, and Shorts.

## 🧠 The Methodology
This system uses **Layered Prompting** to ensure the output isn't just "text," but a production-ready blueprint.
* **Psychological Framework:** Uses PAS (Problem-Agitation-Solution).
* **Hook Variation:** Generates 3 distinct "Scroll-Stoppers" for A/B testing.
* **Native Tone:** Strips away corporate jargon in favor of "Human-to-Human" language.

## 📁 Case Study: The "Glow-Up" Travel Mirror
**Status:** Production Ready
**Target:** Gen Z / Millennial Travelers
### **The Hooks (The Scroll-Stoppers)**
* **The "Relatable Struggle" Hook:** *"POV: You’re trying to get ready in a hotel room with literally zero light."*
* **The "Secret Weapon" Hook:** *"I found the ultimate hack for anyone who does their makeup in the car or on the go."*
* **The "Comparison" Hook:** *"Stop guessing if your foundation is blended. You need to see the difference this makes."*

### **The Script (30-Second Edit)**
| Time | Visual Scene | Audio/Voiceover |
| :--- | :--- | :--- |
| **0:00-0:03** | **Hook:** Close up of user looking frustrated in dim light. | "I am officially done with bad hotel lighting ruining my makeup." |
| **0:03-0:12** | **Problem:** User shows patchy bronzer in a regular mirror. | "I thought I looked fine, then I stepped outside and... yikes." |
| **0:12-0:22** | **Solution:** User unboxes the Glow-Up Mirror, clicks it on. | "Then I got the Glow-Up Travel Mirror. It’s ultra-slim and mimics actual daylight." |
| **0:22-0:30** | **CTA:** User doing a final "check" in the mirror, smiling. | "Seriously, if you travel, grab this. Link in my bio for 20% off today!" |
